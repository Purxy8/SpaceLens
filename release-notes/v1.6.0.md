# SpaceLens 1.6.0

SpaceLens 1.6 is the performance release. It substantially accelerates native and Full access scans, keeps the interface responsive while results arrive, and adds a safe NTFS Quick Refresh for subsequent whole-drive scans.

## Much faster scanning

- Extended Windows directory records provide file IDs, allocated sizes, and reparse tags in buffered native batches without reopening ordinary files.
- Full access transport now uses pooled frames and bounded span-based encoding/decoding instead of per-file streams and temporary path arrays.
- Directory validation, classification, path containment, progress reporting, cache loading, and initial view construction perform less repeated work.
- Scan batches transfer ownership without cloning, and cloud/reparse entries no longer force an extra filesystem query when Windows exposes their tag directly.

## NTFS Quick Refresh

- A completed full-access scan of an entire local NTFS drive stores a validated change-journal checkpoint.
- The next Administrator scan offers either a Fast NTFS refresh or a Complete rescan. Fast refresh reads closed-file NTFS changes and reopens every changed live file by its stable volume/file identity.
- Directory structure changes, hard links, alternate streams, expired/recreated journals, unsupported record versions, ambiguous identities, or path-containment uncertainty automatically trigger a complete full-access scan in the same helper.
- NTFS can coalesce repeated writes while a file stays continuously open, so the interface explains this limitation and always keeps the Complete rescan choice for exact freshly rebuilt results.
- Cleanup remains in the ordinary unelevated UI and still revalidates the selected path and identity immediately before using the Windows Recycle Bin.

## Smoother interface and safer caching

- Live progress is coalesced so a fast scanner cannot flood the UI message queue.
- Results appear as soon as the view is ready while cache compression safely finishes in the background.
- Rapid filter and sort requests cancel/coalesce instead of running competing full-list builds.
- New v7 caches store bounded NTFS checkpoint metadata and retain v6, v5, v4, and v3 compatibility.
- Status text now separates scan, view, and cache time and clearly distinguishes Quick scan, Full access, NTFS refresh, and automatic fallback.

## Download

- **Recommended:** `SpaceLens-Setup.exe` installs or upgrades SpaceLens and maintains Desktop/Start Menu shortcuts.
- **Portable:** `SpaceLens.exe` runs directly without installation.

This release is not yet Authenticode-signed while the SignPath Foundation application is awaiting approval. Windows may therefore show an unknown-publisher or SmartScreen warning. Download only from the official GitHub Releases page and use the accompanying `.sha256` files for manual verification.
