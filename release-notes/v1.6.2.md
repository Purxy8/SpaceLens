# SpaceLens 1.6.2

SpaceLens 1.6.2 fixes large games and applications being easy to miss when their size is spread across thousands of smaller files. The scanner could already index those files, but the **Largest files** tab intentionally retained only 10,000 individual rows and did not show a recursive folder total.

## Folder and game totals

- Added a **Folders & games** tab that totals every matching indexed file, including files that do not appear in the 10,000-row Largest files table.
- Added a default **Detected games & apps** view for Steam, Xbox, Epic, GOG, EA, Ubisoft, Riot, WindowsApps, Program Files, and common drive-root game libraries.
- Added **Browse scan root**, Back, double-click drill-down, and independent folder sorting.
- Category, media, text search, and size sorting remain combined. Selecting a cache category or a search term no longer loses the active game grouping.
- Folder deletion is intentionally unavailable. Open the installation in Explorer, use the game's uninstaller, or inspect individual files before sending anything to the Recycle Bin.

## Accuracy and performance

- Folder rows show both unique physical allocation (**Size on disk**) and logical file lengths (**File sizes**). Logical size is the default sort so compressed, sparse, cloud, and hard-linked files are not silently hidden by physical-attribution order.
- Shared hard links and allocation estimates are marked explicitly.
- Install-root detection uses a bounded recent-root cache and avoids expensive marker parsing for unrelated personal/download files.
- Saved-scan format v8 reclassifies older cached paths and safety levels automatically. Run **Scan now** when the underlying files may have changed.
- Scan completion now calls out inaccessible or linked coverage gaps more clearly.

## Security and reliability

- Ordinary junctions and symbolic links remain excluded. Cloud-placeholder traversal is guarded against path replacement and requires a stable same-volume identity strictly inside the selected root.
- Whole-volume scanner shortcuts are limited to verified local DOS drives; UNC/network shares use the normal containment checks.
- Cleanup candidates exclude application-owned and protected files even when their names resemble caches.
- Release builds, security self-tests, and regression tests cover an 80 GB game split across 20,000 files, 200,000-file folder views, exact-game-root scans, filter composition, hard links, UNC roots, and legacy caches.

## Updating

SpaceLens 1.6.1 users can install this release through **Check for updates** because 1.6.2 uses the same pinned update-verification key. Users on 1.6.0 or earlier must first perform the documented manual 1.6.1 trust bootstrap from the official GitHub release.

## Download and signing status

- **Recommended:** `SpaceLens-Setup.exe` installs or upgrades SpaceLens and maintains Desktop/Start Menu shortcuts.
- **Portable:** `SpaceLens.exe` runs directly without installation.

These 1.6.2 binaries are **not Authenticode-signed**. Windows may show an unknown-publisher or SmartScreen warning. Download only from the official `Purxy8/SpaceLens` GitHub Releases page, verify the accompanying SHA-256 files, and see the [SpaceLens code-signing policy](https://github.com/Purxy8/SpaceLens/blob/main/CODE_SIGNING_POLICY.md).
